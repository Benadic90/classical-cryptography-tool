Classical Cryptography Tool is a beginner-friendly cybersecurity project that demonstrates message encryption and decryption using Caesar Cipher and Vigenère Cipher techniques.

This project helps in understanding the fundamentals of:

Encryption
Decryption
Secret keys
Plaintext and ciphertext
Classical cryptography concepts

It is designed for students and beginners who want to explore basic cryptography and cybersecurity concepts using Python.

Features
Caesar Cipher
Encrypt messages using a shift value
Decrypt encrypted messages
Simple substitution-based encryption
Vigenère Cipher
Encrypt messages using a keyword
Decrypt messages using the same keyword
More secure than Caesar Cipher

Learning Outcomes

By building this project, I learned:

Basics of classical cryptography
How encryption and decryption work
Difference between plaintext and ciphertext
Python string manipulation
Basic cybersecurity concepts
